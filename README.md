# Uploader Service

## Description

REST API Related to Uploader Service

## Features

- [x] Config Service ([@nestjs/config](https://www.npmjs.com/package/@nestjs/config)).
- [x] File uploads. Support local and Amazon S3, Cloud Storage drivers.
- [x] Swagger.
- [x] E2E and units tests.
- [x] Docker.

## Installation
```bash
git clone --depth 1 https://github.com/rezwanul-haque/uploader.git
cd uploader/
cp .env.example .env
```

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
